# Ratan Mahal Resort Website

## Project Setup
This is a Next.js project using Tailwind CSS.

### Prerequisites
- Node.js (v18 or higher recommended)
- npm

### Installation
If you haven't installed dependencies yet:
```bash
npm install
```

### Running Locally (Development)
To start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Building for Production
To build the optimized application:
```bash
npm run build
```

To start the production server after building:
```bash
npm start
```

## Troubleshooting

### Error: `ENOENT: no such file or directory, uv_cwd`
This happens if your terminal is open in a folder that was deleted.
**Fix:**
1. Go back one step:
   ```bash
   cd ..
   ```
2. Or navigate to the project folder again:
   ```bash
   cd "/Users/shiva/Ratan mahal"
   ```
